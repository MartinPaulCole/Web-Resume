RENESAS MICRO
Utilising Renesas M16C Processors to control oven temperature and graphical user displays.
The main controller was located in the oven core and performed PID control of the oven
temperature based on the cooking function selected by the user.

A smaller slave M16C processor was used to drive a separate LCD display unit using DMA
and a Micrium graphics library to feed user functions and preferences to the main unit via
USART comms.

I implemented a bare-metal scheduler to facilitate the software system.

The IDE used was HEW and KD30 and E8 on-chip debugger.
