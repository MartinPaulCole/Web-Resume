Aloha Protocol
ALOHA is basically a multiple access protocol which describes how all the terminals can access a medium without interfering at all with one another or even colliding. It operates at the data-link layer. Roberts created a protocol in 1972 that would double the capacity of ALOHA.

Slotted Aloha Protocol
In Slotted Aloha, the channel time is split into specific time slots. Data can only be sent at the start of a slot. If a station misses its slot, it has to wait for the next one. This makes collisions less likely.

Random Access
Random access protocols refer to communication protocols used in computer networks. Hence, each device has an equal privilege to access the network and transmit data. These protocols allow nodes to send and receive data without taking turns or waiting for permission.
