SCADA Radio
Following a break from telecoms of a few years I joined a company developing a new mulit-point SCADA radio to complement their exisitng point to point high-speed data radio products. This was a new challenge for the company and my role was to optimise channel throughput implementing the driver, MAC and network layers of the software protocol stack.

Radio Platform
VHF and UHF band, point to multipoint radios with 12.5KHz and 25KHz channels, FFSK, 4QAM, 16QAM and 64QAM modulation and AES encryption.

Aloha Protocol
The first stage of this project was to create a driver for a CML RF modem. (ref. CML wireless data) I implemented this using a state machine scheduling packets of up to 256 bytes in and out of the modem TX/RX buffer. This yielded 18% thoughput success due to collisions.

Carrier Sense
The second stage of our project was to use RF carrier sense feedback to allow transmission back-off when detecting the channel is busy. This was a feature provided by the modem. (ref. Carrier Sense Multiple Access) This doubled our throughout to 36%.

Slotted Aloha Protocol
Migrating from an RF Mesh RF Mesh Network to a Base Station controlled Star Network I introduced a base station synchronisation message at 10ms intervals allowed us to reduce the length of the random access messages by a factor of more than 10 (depending on channel width). This increased measured throughput to 80%

Collision Detection - Error Correction
Our project migrated from CML modem to XILINX FPGA implementation which allowed us to increase data rates via muliple levels of Quadrature Amplitude Modulation. Forward Error Correction algorithms were implemented to counteract packet loss due to the increased error rates at longer ranges. (Ref. Collision Detection)

MAC layer and Network
I added a MAC layer on top of the modem driver to allow addressing of units and basic command response requests, retries and back-offs. On top of this I implemented a network layer to allow routing of packets through more than one base station in the case of repeater networks.
