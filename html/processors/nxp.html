NXP Micro
The LPC5460x is a family of ARM Cortex-M4 based microcontrollers for embedded applications featuring a rich peripheral set with very low power consumption and enhanced debug features.

The use of this NXP processor is outlined in EXM Project.

MCUXpresso (Elcipse based) was used for debuggimg.

The LPC54606 has Ten Flexcomm serial interface peripherals. Each Flexcomm Interface can be selected by software to be a USART, SPI, or I2C interface. Two Flexcomm Interfaces also include an I2S interface. Each Flexcomm Interface includes a FIFO buffer that supports USART, SPI, and I2S if supported by that Flexcomm Interface. By ustilising this we were able to implement felxible USART (RS232, RS485) 8 and bit communications to both new and legacy peripherals.

It has a 12-bit, 12-channel, Analog-to-Digital Converter which was used to monitor actuator signals which are critical to the safe operation of the system being controlled.

The SysTick timer is an integral part of the Cortex-M4. The SysTick timer is intended to generate a fixed 10 millisecond interrupt for use by an operating system or other system management software. Since the SysTick timer is a part of the CPU, it facilitates porting of software by providing a standard timer that is available on ARM Cortex-based devices. This SysTick was used to schedule up to 30 run-to-completion tasks at varying rates depending on their priority.
