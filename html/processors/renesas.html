Renesas Micro
Renesas M16C Processors were used while working on a ground-up oven controller project (Ref. Projects - Oven Controller).

A master processor controlled the oven cavity core temperature using a variable PID algorithm based on the user selected oven function. It contained the main scheduler and IO drivers to switch cavity elements.

A smaller slave M16C processor was used to drive a separate LCD display unit using DMA and a Micrium graphics library to feed user functions and preferences to the master unit via USART comms.

I implemented a bare-metal scheduler to facilitate the software system.

The IDE used was HEW and KD30 and E8 on-chip debugger.
