Atmel Micro
My fist experience of ATMEL processors is outlined in AMR Project. AVR Studio was used for debugging. The processor was an ATMega256
The code base was already in existence so my role was to modify it as per aforementioned project.

My second experience of ATMEL processors is outlined in EXM Project. Microchip Studio was used for debugging. The processor was an AVR128DA48 An exising code base had been written for our fire detection sensors team. This was trimmed and modified to implement the USART protocol which communicated with the master extinguishing module controller. It also drives GPIO to an ASIC implementd in-house to communicate to the main fire panels via the MX loop. (ref. CML wireless data)
